# Endpoints (Generados)

Esta sección se genera automáticamente con `knuckleswtf/scribe`.

Acciones sugeridas
1) Publicar configuración de Scribe si no existe: `php artisan vendor:publish --provider="Knuckles\\Scribe\\ScribeServiceProvider" --tag=scribe-config`.
2) Documentar rutas con anotaciones y/o respuestas de ejemplo.
3) Generar: `php artisan scribe:generate`.

Salida
- HTML/Markdown en `public/docs` o carpeta configurada.
