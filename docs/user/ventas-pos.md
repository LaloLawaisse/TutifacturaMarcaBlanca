# Ventas / POS

Propósito: operar ventas en mostrador y facturación rápida.

Flujos
- Venta rápida (POS): alta productos, pagos mixtos, factura.
- Venta tradicional: presupuesto → pedido → factura → remito.

Funciones clave
- Cajas y arqueos, devoluciones, descuentos, listas de precios.

Verificación
- Ticket/Factura emitida y stock descontado.
