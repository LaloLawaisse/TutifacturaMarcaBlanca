# Facturación AFIP

Propósito: configuración y emisión de comprobantes electrónicos en Argentina.

Requisitos
- CUIT, cert/clave digitales vigentes o credenciales homologación.
- Puntos de venta habilitados en AFIP.

Configuración
- Cargar credenciales AFIP en `Configuración → Impuestos/AFIP`.
- Definir talonarios, tipos de comprobante y correlativos.

Emisión
1) Crear venta en POS o ventas.
2) Seleccionar tipo de comprobante (A/B/C, Nota Crédito/Débito).
3) Confirmar y registrar CAE.

Verificación
- CAE válido y PDF con código QR.

Problemas comunes
- Rechazo AFIP: revisar alícuotas, condición IVA, tope de monto.
