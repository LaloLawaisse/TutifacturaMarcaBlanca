# Onboarding

Propósito: alta inicial de negocio y usuarios en Tutifactura.

Requisitos
- Credenciales de administrador inicial.
- Datos fiscales del negocio.

Procedimiento (resumen)
1) Registro de negocio: `GET /business/register` (interfaz web).
2) Verificación de email/usuario.
3) Configuración de locales y series de comprobantes.
4) Alta de usuarios y perfiles.

Verificación
- Acceso a panel principal y emisión de comprobante de prueba.

Problemas comunes
- Usuario ya registrado: validar disponibilidad de email/usuario.
